# Sistema de Control de Inventario - Systeminv

Proyecto integral documentado de inicio a fin, desarrollado para optimizar la gestión de stock en un restaurante pyme.  
Incluye análisis de la empresa, levantamiento de requerimientos, estudio de factibilidad, diseño UML, construcción de prototipo funcional, pruebas y conclusiones.
